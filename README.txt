Come eseguire il programma:
	+ Da terminale raggiungere la cartella contentente client.ol (oppure server.ol)
	+ digitare "jolie client.ol" oppure "jolie server.ol"
	+ Per avviare il server con una porta differente da quella di default (8001) digitare:
		"jolie -C SERVER_LOCATION=\"socket://localhost:numeroPorta\" server.ol"

Se si volessere lanciare più server e/o più client è sufficiente copiare la cartella
Client o Server.
